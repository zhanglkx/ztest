💡 什么是 CSS Grid？

CSS Grid Layout 是一个二维布局系统，它允许你同时在行和列两个维度上控制网页布局。与Flexbox（一维布局）不同，Grid可以轻松创建复杂的网格结构。

📚 核心术语解析
🏗️ 网格容器 (Grid Container)
设置了 display: grid 或 display: inline-grid 的元素，它是所有网格项目的父元素。
📦 网格项目 (Grid Item)
网格容器的直接子元素。它们会自动成为网格项目，可以被放置在网格的任意位置。
📏 网格线 (Grid Line)
构成网格结构的分界线。分为行线（水平）和列线（垂直），用数字或名称标识。
🔲 网格轨道 (Grid Track)
两条相邻网格线之间的空间。可以是行轨道（行与行之间）或列轨道（列与列之间）。
🎯 网格单元 (Grid Cell)
网格中的最小单位，由相邻的行线和列线围成的空间，类似表格中的单元格。
🗺️ 网格区域 (Grid Area)
由四条网格线围成的矩形区域，可以包含一个或多个网格单元。

网格线编号规则：

  • 列线从左到右编号：1, 2, 3, 4...
  • 行线从上到下编号：1, 2, 3, 4...
  • 也可以使用负数从右/下边开始编号：-1, -2, -3...
  • 网格线可以命名，便于理解和维护
🔧 创建你的第一个网格

让我们从最简单的例子开始，创建一个3×2的网格：

/* HTML */
<div class="my-grid">
    <div class="item">A</div>
    <div class="item">B</div>
    <div class="item">C</div>
    <div class="item">D</div>
    <div class="item">E</div>
    <div class="item">F</div>
</div>

/* CSS */
.my-grid {
    display: grid;                    /* 启用Grid布局 */
    grid-template-columns: 1fr 1fr 1fr;  /* 3列，等宽 */
    grid-template-rows: 100px 100px;     /* 2行，固定高度 */
    gap: 20px;                       /* 网格间距 */
}

.item {
    background: #667eea;
    color: white;
    display: flex;
    align-items: center;
    justify-content: center;
    border-radius: 8px;
}

✨ 关键知识点总结
1️⃣ 启用Grid
使用 display: grid 将元素设置为网格容器
2️⃣ 二维布局
可以同时控制行和列的布局，比Flexbox更强大
3️⃣ 自动排列
子元素会自动按照网格结构排列，无需额外定位
4️⃣ 网格线系统
使用数字或名称来标识网格线，便于精确定位
