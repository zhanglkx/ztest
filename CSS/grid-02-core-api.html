🔧 第2课：CSS Grid 核心API

掌握Grid布局的核心属性和方法

🎯 display: grid - 启用网格布局

要使用Grid布局，首先需要在容器上设置 display: grid 或 display: inline-grid。

🏗️ display: grid
display: grid;

创建一个块级网格容器

📦 display: inline-grid
display: inline-grid;

创建一个行内网格容器

.container {
    display: grid;  /* 启用网格布局 */
}
📏 grid-template-columns - 定义列

这是最重要的Grid属性之一，用来定义网格的列数和每列的大小。

🎮 交互式演示

1
2
3
grid-template-columns: 1fr 2fr 1fr;
🔢 fr 单位 (fraction)
grid-template-columns: 1fr 2fr 1fr;

表示可用空间的分数。上例中第二列是第一、三列的两倍宽。

📐 固定单位
grid-template-columns: 200px 150px 100px;

使用px、em、rem等固定单位定义列宽。

📊 百分比
grid-template-columns: 25% 50% 25%;

基于容器宽度的百分比。

🔄 repeat() 函数
grid-template-columns: repeat(4, 1fr);

等同于 1fr 1fr 1fr 1fr，用于创建重复的列。

🤖 auto 关键字
grid-template-columns: auto 1fr auto;

根据内容自动调整大小。

📏 minmax() 函数
grid-template-columns: minmax(100px, 1fr);

定义大小的最小值和最大值范围。

📐 grid-template-rows - 定义行

grid-template-rows 的用法与 grid-template-columns 类似，用于定义行的数量和高度。

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);  /* 3列等宽 */
    grid-template-rows: 100px 150px 100px;  /* 3行不等高 */
    gap: 15px;
}
💡 提示
如果不指定 grid-template-rows，行高会根据内容自动调整。
🎨 grid-template 简写属性

grid-template 是 grid-template-rows 和 grid-template-columns 的简写形式。

/* 完整写法 */
.grid-container {
    grid-template-rows: 100px 150px;
    grid-template-columns: 1fr 2fr 1fr;
}

/* 简写形式 */
.grid-container {
    grid-template: 100px 150px / 1fr 2fr 1fr;
    /*             行高度      /   列宽度    */
}
✏️ 语法规则
grid-template: [rows] / [columns];

行和列的定义用斜杠 (/) 分隔。

🎯 实际示例
grid-template: repeat(3, 1fr) / 200px 1fr;

3行等高，2列（固定宽度+弹性宽度）。

🔧 实战技巧与最佳实践
⚡ 性能优化

尽量使用 fr 单位而不是百分比，fr 单位性能更好且更灵活。

📱 响应式设计

结合媒体查询，在不同屏幕尺寸下使用不同的列数和大小。

🎨 设计一致性

使用 repeat() 函数创建规律的网格，保持视觉一致性。

🔍 调试技巧

使用浏览器开发者工具的Grid检查器查看网格结构。

🚀 进阶技巧

可以混合使用不同的单位类型，如：grid-template-columns: 250px 1fr minmax(200px, 300px)，这样可以创建更灵活的布局。
