↔️ 分别设置
gap: 10px 30px;

🔄 单独控制
row-gap: 10px; column-gap: 30px;

分别控制行间距和列间距

🚀 默认项目对齐 - justify-items 与 align-items

为所有网格项目设置默认的对齐方式。

/* 为容器内所有项目设置默认对齐 */ .grid-container { justify-items: center; /* 所有项目水平居中 */ align-items: center; /* 所有项目垂直居中 */ /* 简写形式 */ place-items: center; /* 等同于上面两行 */ } /* 单个项目仍可以覆盖默认设置 */ .special-item { justify-self: start; /* 覆盖默认的center */ }
💡 对齐属性层级关系

容器设置： justify-items/align-items → 设置所有项目的默认对齐
项目设置： justify-self/align-self → 覆盖容器的默认设置

📐 对齐值说明

start: 起始位置对齐
end: 结束位置对齐
center: 居中对齐
stretch: 拉伸填充（默认值）

🔄 space 分布

space-between: 两端对齐
space-around: 环绕分布
space-evenly: 平均分布

🎨 实用技巧与最佳实践
🎯 居中技巧

使用 place-items: center 可以让所有网格项目都居中，这是最简单的居中方法。

📱 响应式间距

在移动端可以减少gap值，桌面端增大gap值，提升用户体验。

⚡ 性能考虑

使用gap替代margin设置间距，gap不会影响盒模型计算，性能更好。

🔍 调试工具

浏览器开发者工具的Grid检查器可以直观地看到对齐效果。

/* 实用的居中布局 */ .center-layout { display: grid; place-items: center; /* 项目居中 */ place-content: center; /* 网格居中 */ min-height: 100vh; /* 全屏高度 */ } /* 响应式间距 */ .responsive-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 15px; } @media (min-width: 768px) { .responsive-grid { gap: 30px; } }
